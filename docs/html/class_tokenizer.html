Detailed Description

Object used to obtain tokens from a string according to an finite automata and a keyword set.

Constructor & Destructor Documentation

Tokenizer::Tokenizer ( )

Constructs a Tokenizer object. The object constructed does not have a set automata or keyword set, so the user should remember to set them before calling getToken().

void Tokenizer::setAutomata ( AbstractAutomata *  automata)

Sets the automata used on this tokenizer.

Sets the automata to be used when tokenizing the given input string. The automata passed should be dynamically allocated. If a previous automata had been set, it will be deleted.

Parameters
